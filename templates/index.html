Student CRUD API

A simple Flask-based API for managing students, featuring AI-powered summaries with Ollama.

API Endpoints:
  • GET /students – List all students
  • POST /students – Add a new student
  • GET /students/<id> – Get a student by ID
  • PUT /students/<id> – Update a student by ID
  • DELETE /students/<id> – Delete a student by ID
  • GET /students/<id>/summary – Get AI summary for a student

For API usage, use tools like Postman or curl.

Backend: Flask + Python
AI Summaries: Ollama (llama3.1:latest)
